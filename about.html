About this project

This project reflects on online personality quizzes.

Many quizzes claim to reveal deep truths, but often rely on vague questions and flexible interpretations.

This quiz mimics that structure but produces simple results, inviting users to question how meaning is created.

Development Log

This project started from the observation that people often take personality quizzes and still feel the results are accurate, even when the questions are unclear.

The quiz uses a simple structure with multiple-choice questions shown one at a time. It is designed to feel like a typical personality test.

However, the final result is actually determined only by the last question. The earlier questions create a sense of buildup, but do not affect the outcome.

The project is built with HTML, CSS, and JavaScript, using localStorage to keep track of answers and progress.

Overall, the project explores how easily meaning can be constructed through structure, language, and expectation.
